# API для Clarity Project

[Опис API](README.md)

### Автотранспорт у власності

## vehicles.list
Метод повертає список автотраспорту у власності фізичної чи юридичної особи 
за її кодом РНОКПП чи ЄДРПОУ.

##### Приклад запиту:
https://clarity-project.info/api/vehicles.list/25412361?key=xxx,
де **25412361** - код ЄДРПОУ або РНОКПП суб'єкта.

### Формат відповіді
Повертається список автомобілів та їх останніх реєстрацій дій: 
```json
{
  "vehicles":
  [
    {
      "operCode": 350,
      "operName": "ПЕРЕРЕЄСТРАЦIЯ У ЗВ`ЯЗКУ ЗI ЗМIНОЮ НАЗВИ ОРГАНIЗАЦIЇ",
      "depName": "ТСЦ 3244",
      "brand": "ISUZU",
      "model": "NQR 71P",
      "makeYear": "2008",
      "capacity": 4570,
      "dreg": "2021-03-25",
      "color": "БІЛИЙ",
      "kind": "ВАНТАЖНИЙ",
      "body": "ФУРГОН ІЗОТЕРМІЧНИЙ",
      "purpose": "СПЕЦІАЛІЗОВАНИЙ",
      "fuel": "ДИЗЕЛЬНЕ ПАЛИВО",
      "ownWeight": 3700,
      "totalWeight": 8000
    }
  ]
}
```

Зверніть увагу: номер автомобіля та vin-код в цих даних відсутні.

У випадку, якщо у суб'єкта відсутній автотранспорт 
або переданий невалідний код — повертається пустий список.
