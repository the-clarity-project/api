# API для Clarity Project

[Опис API](README.md)

### Нерухомість у власності

## realty.list
Метод повертає список автотраспорту у власності фізичної чи юридичної особи
за її кодом РНОКПП чи ЄДРПОУ.

**⚠ ЗВЕРНІТЬ УВАГУ:**
Для доступу до даних Державного реєстру речових прав на нерухоме майно необхідно 
пройти ідентифікацію (це вимога закону).

Користувач-власник API-ключа має пройти ідентифікацію 
[в своєму профілі](https://clarity-project.info/me/identification).

В іншому випадку цей ендпоінт буде повертати помилку.

##### Приклад запиту:
https://clarity-project.info/api/realty.list?key=xxx&code=1234&name=foo,
де **code** - код ЄДРПОУ або РНОКПП суб'єкта,
**name** - ПІБ фізичної особи.

API підтримує такі варіанти запитів:
- юридична особа за її кодом ЄДРПОУ: `code=12345678`, `name` - відсутній;
- фізична особа за її кодом РНОКПП: `code=1234567890`, `name` - відсутній;
- фізична особа за її ПІБ: `name=Шевченко Микола Сергійович`, `code` - відсутній;
